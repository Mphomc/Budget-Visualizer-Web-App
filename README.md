# Budget Visualizer

A simple web app to track your expenses and see them in a pie chart.

---

## Features
- Add expenses with a category and amount.
- See a list of all expenses.
- Remove expenses easily.
- Pie chart updates automatically to show your spending.

---

## Tech Used
- HTML, CSS, JavaScript
- Chart.js for the chart
- Node.js & Express for running locally

---

## How to Run
1. Download or copy the project to your computer.
2. Open a terminal in the project folder.
3. npm init -y
   npm install express
   node server.js
4. Go to browser and type this link =>  http://localhost:3000
